# Roadmap de Skills

## Objetivo
Optimizar el agente general y el equipo con skills reutilizables y ligeras.

## Fase 1 (completada)
- ResearchSkills, SourceValidationSkills
- DocsSkills, SynthesisSkills
- VisualizationSkills
- QualitySkills, RiskSkills
- TriageSkills, WorkflowSkills
- DataNormalizationSkills
- MemorySkills, ToolDecisionSkills

## Fase 2 (siguiente)
- ContextCompressionSkills (compactar historial largo)
- DecisionLogSkills (registro de decisiones)
- ConsistencySkills (coherencia entre agentes)
- GapAnalysisSkills (faltantes vs requerimientos)

## Fase 3 (avanzada)
- CrossCheckSkills (contrastar fuentes)
- OutlierDetectionSkills (validación numérica)
- ExecutiveSummarySkills (resúmenes ejecutivos)
- SecurityScanSkills (detectar secretos/credenciales)

## Recomendación de asignación
- Agente general: todas las skills de síntesis, calidad, memoria y decisión.
- Manager: skills de workflow, triage, calidad y síntesis.
- Researcher: research + source_validation.
- Analyst: data_normalization + visualization (brief).
- Reviewer: quality + risk + decision_log.
